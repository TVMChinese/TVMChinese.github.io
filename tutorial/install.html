安装 TVM¶

作者: Jocelyn Shiue, Chris Hoge

根据你的需求和开发环境，有几种不同的安装 TVM 的方法。包括：

  • 从源码安装

  • 从第三方二进制包安装。

从源码安装¶

从源码安装是安装 TVM 的推荐方法。它将允许你启用特定功能，比如对 GPU 和微控制器（microTVM）的支持、运行时调试器，以及其他功能。如果您想为 TVM 项目贡献代码，那需要用源码安装。完整的说明请参考页面 Install TVM From Source <https://tvm.apache.org/docs/install/from_source.html>。

从二进制包安装¶

你可以方便地从第三方二进制包安装 TVM，以便快速尝试。TLCPack 是一个从 TVM 源码构建二进制包的志愿者社区。它提供了矩阵支持和不同平台上安装的说明，并且具有不同的功能 。请参考 TLCPack <https://tlcpack.ai> 以了解更多。请注意，第三方二进制软件包可能包含与它捆绑的硬件驱动程序的其他许可条款。

Download Python source code: install.py

Download Jupyter notebook: install.ipynb

Relay 基础张量算子¶

此页面包含 tvm.realy 中预先定义的核心张量算子原始数列表。核心张量算子原语涵盖深度学习中的典型工作量。它们可以代表前端框架中的工作内容，并为优化提供基本构建基块。由于深度学习是一个快速发展的领域，因此有可能有些算子不在这里。

注解

本文档将直接列出这些算子在 python 前端中的函数签名。

算子概述¶

级别 1：基础算子

此级别可实现多层的全连接感知器。

tvm.relay.log

数据的点对应对数运算。

tvm.relay.sqrt

数据的点对应平方根运算。

tvm.relay.rsqrt

数据的点对应平方根倒数运算。

tvm.relay.exp

数据的点对应以 e 为底的指数运算。

tvm.relay.sigmoid

数据的点对应 sigmoid 运算。

tvm.relay.add

Numpy 风格的广播加法运算。

tvm.relay.subtract

Numpy 风格的广播减法运算。

tvm.relay.multiply

Numpy 风格的广播乘法运算。

tvm.relay.divide

Numpy 风格的广播除法运算。

tvm.relay.mod

Numpy 风格的广播取模运算。

tvm.relay.tanh

数据的点对应 tanh 运算。

tvm.relay.concatenate

根据给定的轴连接输入张量。

tvm.relay.expand_dims

在 axis 给定的位置插入 num_newaxis 坐标轴。

tvm.relay.nn.softmax

计算 softmax。

tvm.relay.nn.log_softmax

计算 log_softmax。

tvm.relay.nn.relu

修正线性单元。

tvm.relay.nn.dropout

对输入数列使用 dropout 算子。

tvm.relay.nn.batch_norm

批量处理规范化层 （Ioffe and Szegedy, 2014）。

tvm.relay.nn.bias_add

add_bias 算子。

级别 2： 卷积

此级别使用在典型的卷积模型。

tvm.relay.nn.conv2d

2D 卷积。

tvm.relay.nn.conv2d_transpose

二维转置卷积算子。

tvm.relay.nn.conv3d

3D 卷积。

tvm.relay.nn.conv3d_transpose

3D 转置卷积。

tvm.relay.nn.dense

稠密算子。

tvm.relay.nn.max_pool2d

2D 最大池化算子。

tvm.relay.nn.max_pool3d

3D 最大池化算子。

tvm.relay.nn.avg_pool2d

2D 平均池化算子。

tvm.relay.nn.avg_pool3d

3D 平均池化算子。

tvm.relay.nn.global_max_pool2d

2D 全局最大池化算子。

tvm.relay.nn.global_avg_pool2d

2D 全局平均池化算子。

tvm.relay.nn.upsampling

上采样。

tvm.relay.nn.upsampling3d

3D 上采样。

tvm.relay.nn.batch_flatten

BatchFlatten.

tvm.relay.nn.pad

填充

tvm.relay.nn.lrn

该算子将数据作为输入，并进行局部响应归一化。

tvm.relay.nn.l2_normalize

对输入数据执行L2标准化

tvm.relay.nn.bitpack

对于位序列算子的张量包装。

tvm.relay.nn.bitserial_dense

位序列稠密算子。

tvm.relay.nn.bitserial_conv2d

使用位序列计算的 2D 卷积。

tvm.relay.nn.contrib_conv2d_winograd_without_weight_transform

基于 Winograd 算法的 2D 卷积。

tvm.relay.nn.contrib_conv2d_winograd_weight_transform

基于 Winograd 算法对 2D 卷积的转换部分进行加权。

tvm.relay.nn.contrib_conv3d_winograd_without_weight_transform

基于 Winograd 算法的 3D 卷积。

tvm.relay.nn.contrib_conv3d_winograd_weight_transform

基于 Winograd 算法对 3D 卷积的转换部分进行加权。

级别 3：附加数学函数和Transform 算子

此级别是额外的数学函数和transform 算子。

tvm.relay.nn.leaky_relu

该算子将数据作为输入，并代入带泄露的修正线性单元。

tvm.relay.nn.prelu

该算子将数据作为输入，并代入带泄露的修正线性单元。

tvm.relay.reshape

重塑输入数组。

tvm.relay.reshape_like

根据其他张量的形状，对输入张量进行重塑。

tvm.relay.copy

复制张量。

tvm.relay.transpose

对数组进行换维操作。

tvm.relay.squeeze

对数组进行维度压缩。

tvm.relay.floor

数据的点对应向下取整运算。

tvm.relay.ceil

数据的点对应向上取整运算。

tvm.relay.sign

数据的点对应取绝对值运算。

tvm.relay.trunc

数据的点对应舍尾取整运算。

tvm.relay.clip

将 a 中的元素裁剪至 a_min 到 a_max 之间。

tvm.relay.round

数据的点对应四舍五入运算。

tvm.relay.abs

数据的点对应取绝对值运算。

tvm.relay.negative

数据的点对应取相反数运算。

tvm.relay.take

从数组的一条轴上取元素。

tvm.relay.zeros

用0填充数组。

tvm.relay.zeros_like

返回一个与输入数据相同数据类型与尺寸的全零数组。

tvm.relay.ones

用1填充数组。

tvm.relay.ones_like

返回一个与输入数据相同数据类型与尺寸的全一数组。

tvm.relay.gather

在给定的轴上对给定索引的值进行聚合。

tvm.relay.gather_nd

聚合数据的元素或切片，并将其储存到一个尺寸由索引定义的张量里。

tvm.relay.full

用标量填充数组。

tvm.relay.full_like

返回一个与输入数据相同数据类型与尺寸的标量数组。

tvm.relay.cast

转换输入向量的数据类型。

tvm.relay.reinterpret

将输入张量重新解释为指定数据类型。

tvm.relay.split

沿轴按照一个元素或数组对输入张量进行切分。

tvm.relay.arange

在给定范围内返回均匀间隔的值。

tvm.relay.meshgrid

从坐标向量中创建坐标矩阵。

tvm.relay.stack

沿新轴连接一个数组序列。

tvm.relay.repeat

重复数组元素。

tvm.relay.tile

将整个数组重复若干次。

tvm.relay.reverse

在保持数组形状不变的情况下，沿给定的轴反转元素顺序。

tvm.relay.reverse_sequence

将张量内的每个可变长度切片进行翻转。

tvm.relay.unravel_index

将平面索引或平面索引数组转换为坐标数组的元组。

tvm.relay.sparse_to_dense

将稀疏张量表示转换为稠密张量。

**级别 4：广播和降维*

tvm.relay.right_shift

Numpy 风格的广播向右移位运算。

tvm.relay.left_shift

Numpy 风格的广播向左移位运算。

tvm.relay.equal

对于(lhs == rhs)，进行基于广播后的点对应判断。

tvm.relay.not_equal

对于(lhs != rhs)，进行基于广播后的点对应判断。

tvm.relay.greater

对于(lhs > rhs)，进行基于广播后的点对应判断。

tvm.relay.greater_equal

对于(lhs >= rhs)，进行基于广播后的点对应判断。

tvm.relay.less

对于(lhs < rhs)，进行基于广播后的点对应判断。

tvm.relay.less_equal

对于(lhs <= rhs)，进行基于广播后的点对应判断。

tvm.relay.all

沿着给定的轴对布尔数组的元素执行逻辑与计算。

tvm.relay.any

沿着给定的轴对布尔数组的元素执行逻辑或计算。

tvm.relay.logical_and

Numpy 风格的广播逻辑与运算。

tvm.relay.logical_or

Numpy 风格的广播逻辑或运算。

tvm.relay.logical_not

数据的点对应逻辑否运算。

tvm.relay.logical_xor

Numpy 风格的广播逻辑异或运算。

tvm.relay.maximum

Numpy 风格的广播取最大值运算。

tvm.relay.minimum

Numpy 风格的广播取最小值运算。

tvm.relay.power

Numpy 风格的广播幂运算。

tvm.relay.where

根据 condition 的值来决定选择 x 或 y 的元素。

tvm.relay.argmax

沿轴返回最大值的索引。

tvm.relay.argmin

沿轴返回最小值的索引。

tvm.relay.sum

计算给定轴上数组元素的和。

tvm.relay.max

计算给定轴上数组元素的最大值。

tvm.relay.min

计算给定轴上数组元素的最小值。

tvm.relay.mean

计算给定轴上数组元素的平均值。

tvm.relay.variance

计算给定轴上数组元素的方差。

tvm.relay.std

计算给定轴上数组元素的标准差。

tvm.relay.mean_variance

计算给定轴上数组元素的平均值与方差。

tvm.relay.mean_std

计算给定轴上数组元素的平均值与标准差。

tvm.relay.prod

计算给定轴上数组元素的乘积。

tvm.relay.strided_slice

提取数组的分段切片。

tvm.relay.broadcast_to

将标量数组广播至给定的形状并返回，且不改变其数据类型。

级别 5：视觉/图像算子

tvm.relay.image.resize1d

图像 1 维重塑算子。

tvm.relay.image.resize2d

图像 2 维重塑算子。

tvm.relay.image.resize3d

图像 3 维重塑算子。

tvm.relay.image.crop_and_resize

裁剪并调整输入图像的大小。

tvm.relay.image.dilation2d

2D 形态学膨胀。

tvm.relay.vision.multibox_prior

在 data 中，根据 sizes 和 ratios 生成先验框（锚框）。

tvm.relay.vision.multibox_transform_loc

Location transformation for multibox detection

tvm.relay.vision.nms

Non-maximum suppression operations.

tvm.relay.vision.yolo_reorg

Yolo reorg operation used in darknet models.

级别 6：算法算子

tvm.relay.argsort

沿给定轴对数组执行排序操作后，保持数组形状不变，输出排序数组对应的索引数组。

tvm.relay.topk

沿给定的轴，获取输入张量的最大 k 个元素。

级别 10： 临时算子

此级别支持广播算子的反向传播。这是暂时的。

tvm.relay.broadcast_to_like

返回一个与输入数据相同数据类型与尺寸的标量数组。

tvm.relay.collapse_sum_like

返回一个与输入数据相同数据类型与尺寸的标量数组。

tvm.relay.slice_like

按照第二个输出值的形状对第一个输出值进行切分。

tvm.relay.shape_of

获得张量的形状。

tvm.relay.ndarray_size

获得张量的元素数量。

tvm.relay.layout_transform

变换张量的布局。

tvm.relay.device_copy

将数据从源设备复制到目标设备。

tvm.relay.annotation.on_device

Annotates an expression with the device type on which its result should be stored.

tvm.relay.reverse_reshape

Reshapes the input array where the special values are inferred from right to left.

tvm.relay.sequence_mask

Sets all elements outside the expected length of the sequence to a constant value.

tvm.relay.nn.batch_matmul

Compute batch matrix multiplication of tensor_a and tensor_b.

tvm.relay.nn.adaptive_max_pool2d

2D adaptive max pooling operator.

tvm.relay.nn.adaptive_avg_pool2d

2D adaptive average pooling operator.

tvm.relay.one_hot

Returns a one-hot tensor where the locations repsented by indices take value on_value, other locations take value off_value.

级别 11：Dialect 算子

This level supports dialect operators.

tvm.relay.qnn.op.requantize

Requantized operator.

tvm.relay.qnn.op.conv2d

Quantized 2D convolution.
